#challenge-data-science-alura

##Análisis de Datos - Alura Store
###Descripción del Proyecto

Este proyecto tiene como objetivo analizar el desempeño de cuatro tiendas de Alura Store para apoyar la toma de decisiones estratégicas sobre cuál tienda debería vender el Sr. Juan.

El análisis incluye evaluación de ingresos, categorías de productos, calificaciones de clientes, productos más y menos vendidos, costos de envío y distribución geográfica de las ventas.

###Análisis Realizados

✔ Ingresos totales por tienda

✔ Ventas por categoría

✔ Calificación promedio de clientes

✔ Productos más y menos vendidos

✔ Costo de envío promedio

✔ Análisis geográfico con latitud y longitud

###Visualizaciones Generadas

Gráfico de barras (facturación)

Gráfico de línea (calificaciones)

Gráfico de dispersión / mapa de calor (distribución geográfica)

###Conclusión

Tras analizar los datos, se identificó que la Tienda 4 presenta el desempeño general más bajo en comparación con las demás, considerando ingresos, satisfacción del cliente y costos logísticos.

Por lo tanto, se recomienda que el Sr. Juan considere vender dicha tienda.
